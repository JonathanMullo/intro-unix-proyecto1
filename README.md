# First Term Final Project - Introduction to Unix

## Description
This project covers fundamental Unix/Linux concepts including disk encryption, 
password management, cryptographic hashing, secure email, and Git version control.

## Objectives
- Install and configure Debian 13 with LVM and full disk encryption
- Manage passwords securely using KeePassXC and Shannon entropy theory
- Analyze cryptographic hash functions and perform cracking with Hashcat
- Send secure signed emails using GPG and ProtonMail
- Manage a Git repository with GPG-signed commits on GitHub
